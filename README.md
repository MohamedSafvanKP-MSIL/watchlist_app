## Features

- **Home Screen**: Home Screen with Bottom Navigation Bar.

- **Search Bar**:
    - Contains default symbols
    - Filter and display matching symbols while typing
- **List of Symbols**:
    - Long press a symbol to enter edit mode
    - Rearrange symbols in edit mode
